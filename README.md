 Smart Plant System v2.0 (Pro/Industrial Grade) – Descriere ProiectDescriere Generală„Smart Plant System v2.0” este un sistem IoT autonom, stabil și fault-tolerant, conceput pentru monitorizarea precisă și automatizarea irigării plantelor. Folosind arhitectura Dual-Core a unui ESP32 și sistemul de operare de timp real FreeRTOS, dispozitivul rulează procese paralele fără blocaje.Față de sistemele hobby, v2.0 integrează hardware fail-safes (protecții electrice, izolări galvanice) și senzori de precizie industrială pentru a garanta funcționarea pe termen lung nesupravegheată. Sistemul expune un dashboard web local în timp real, trimite alerte Telegram și utilizează NTP cu compensare de fus orar pentru udări programate exact.⚙️ Arhitectură Hardware & Fail-SafesIzolare și Protecție: Alimentare ramificată pentru microcontroller și sarcinile inductive (pompă). Releul este izolat prin optocuplor, iar pompa este protejată de o diodă Flyback pentru a preveni vârfurile de tensiune (Back-EMF).HMI Local (Human-Machine Interface): Un ecran OLED permite diagnosticarea la fața locului (IP, status net, erori) fără a depinde de un browser web.Protecție la funcționare în gol (Dry-Run): Un senzor cu flotor în rezervor blochează pompa dacă nu există apă, prevenind distrugerea acesteia și emițând o alertă de sistem.🧵 Model Software (C++ Modern & FreeRTOS)Thread Safety: Utilizarea mecanismelor de tip Mutex (RAII Lock Guards) pentru a proteja memoria partajată între task-uri (Race Condition prevention).Filtrare Semnal: Filtru software de tip Moving Average aplicat peste datele senzorului de sol pentru a elimina zgomotul specific convertorului ADC al ESP32.Metaprogramare: Sistem configurabil integral la nivel de compilator prin directive preprocesor (#define), permițând activarea/dezactivarea modulelor fără a rescrie logica core.🛒 Bill of Materials (BOM)Iată lista de componente pentru a construi versiunea hardware blindată.

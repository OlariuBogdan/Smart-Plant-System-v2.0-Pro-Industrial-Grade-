# Smart-Plant-System-v2.0-Pro-Industrial-Grade-
Smart Plant System v2.0” este un sistem IoT autonom, stabil și fault-tolerant, conceput pentru monitorizarea precisă și automatizarea irigării plantelor. Folosind arhitectura Dual-Core a unui ESP32 și sistemul de operare de timp real FreeRTOS, dispozitivul rulează procese paralele fără blocaje.
